About

Return to Pic Poet

Pic Poet is a simple web app that generates poems from images. It uses a pre-trained neural network to identify objects in the image (Google's Cloud Vision AI), and then uses a pre-trained language model (OpenAI, GPT-3.5-turbo) to generate a poem based on the identified objects. This project is deployed on Cloud Run, a serverless platform that allows for easy scaling and deployment.

Pic Poet was created by Stuart Rimel as a final project for Internet, Web, and Cloud Systems at Portland State University.
